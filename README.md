## Getting Started

### 1. Clone the Repository

```bash
git clone https://kiran-pokhrel.com.np
```

### 2. Install Dependencies

```bash
npm install
```

### 3. Run the Development Server

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

The application will run at:
`http://localhost:5173`
